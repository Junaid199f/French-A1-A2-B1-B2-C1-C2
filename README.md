# 🇫🇷 Apprendre le français — Préparation DELF / DALF

Ressources d'étude organisées du niveau A1 au niveau C2, sur le modèle du pack allemand fourni, mais adaptées au français et aux examens DELF / DALF.

Les niveaux A1 à B1 sont rédigés en mode bilingue (français + anglais) pour faciliter l'entrée dans la langue. Les niveaux B2 à C2 privilégient l'immersion en français, avec des traductions anglaises limitées aux tableaux de vocabulaire.

| [B2](./B2/) | DELF B2 tout public | Intermédiaire supérieur / Upper intermediate | 🇫🇷 |

## Remarques

- Ce pack est un support de préparation structuré, pas un remplacement des manuels officiels ni des sujets authentiques.
- Les formats d'épreuves suivent la logique DELF / DALF et les durées les plus courantes, mais il faut toujours vérifier les consignes auprès du centre d'examen.
- Les exemples sont originaux et conçus pour la pratique.
