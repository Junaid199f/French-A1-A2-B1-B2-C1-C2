# A1 Expression écrite / Writing

## Tâches fréquentes

- Remplir un formulaire d'inscription. / Fill in a registration form.
- Écrire une carte postale à un ami. / Write a postcard to a friend.

## Modèle

```text
Bonjour Sara,
Je suis à Paris pour trois jours. La ville est très belle. Je visite un musée demain et je mange dans un petit restaurant ce soir. Il fait beau. À bientôt.
Amicalement,
Nora
```

## Conseils
- Respectez le sujet, écrivez des phrases courtes et vérifiez les informations personnelles.
