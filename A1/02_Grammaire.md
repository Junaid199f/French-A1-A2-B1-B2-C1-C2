# A1 Grammaire / Grammar

Points clés de grammaire pour ce niveau. Key grammar points for this level.

## 1. Les articles définis et indéfinis

Résumé / Summary: le, la, les / un, une, des.
- le livre, la table, les amis
- un café, une chaise, des étudiants

## 2. Le verbe être et le verbe avoir

Résumé / Summary: je suis, tu es / j'ai, tu as.
- Je suis à Paris.
- Nous avons deux billets.

Résumé / Summary: mon, ma, mes / ton, ta, tes.
