# B1 Écoute / Listening

## Support : Chronique radio : Le télétravail

### Transcription

- Animateur : Aujourd'hui, nous parlons du télétravail.
- Invitée : Pour beaucoup de salariés, il offre plus de souplesse et moins de temps de transport.
- Animateur : Mais certains se sentent isolés.
- Invitée : Oui, surtout quand l'entreprise n'organise pas assez de réunions d'équipe.
- Animateur : Finalement, un modèle hybride semble préférable pour de nombreuses professions.

### Questions

- Le télétravail réduit parfois le temps de déplacement. — Réponse : **Oui**
- Tout le monde préfère travailler uniquement à distance. — Réponse : **Non**
- L'invitée soutient un modèle hybride. — Réponse : **Oui**

### Conseils
- Notez l'idée générale puis deux ou trois détails précis.
